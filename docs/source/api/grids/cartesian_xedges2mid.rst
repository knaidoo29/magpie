.. function:: magpie.grids.xedges2mid(xedges)

    Returns the mid-point of a uniform grid from the grid edges.

    :Parameters:
      xedges : array
          The edges of a uniform grid along one axis.

    :Returns:
      xmid : array
          The mid-point of a uniform grid.
