
magpie.randoms.random_draw
^^^^^^^^^^^^^^^^^^^^^^^^^^


.. function:: magpie.randoms.random_draw(sample, size)

    Draws a random sample from an input function, the algorithm ensures there
    can be no repeats.

    :Parameters:
      sample : array
          Input sample data.
      size : int
          Size of the random draws.

    :Returns:
      randsamp : array
          Random subsample.
