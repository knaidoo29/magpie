.. function:: magpie.randoms.randoms_sphere_r(size[, r_min=0., r_max=1.])

    Random radial points for a segment of a sphere (default will give randoms within a unit sphere).

    :Parameters:
      size : int
          Number of randoms to generate.
      r_min : float
          Minimum radius.
      r_max : float
          Maximum radius.

    :Returns:
      r : array
          Random r.
