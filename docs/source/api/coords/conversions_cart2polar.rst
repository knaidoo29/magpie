
magpie.coords.cart2polar
^^^^^^^^^^^^^^^^^^^^^^^^

.. function:: magpie.coords.cart2polar(x, y[, center=[0., 0.]])

    Returns the polar coordinates for a given set of cartesian coordinates.

    :Parameters:
      x : array
          x coordinates.
      y : array
          y coordinates.
      center : list
          Center point of polar coordinate grid.

    :Returns:
      r : array
          Radial coordinate.
      phi : array
          Phi coordinate.
