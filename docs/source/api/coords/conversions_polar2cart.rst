
magpie.coords.polar2cart
^^^^^^^^^^^^^^^^^^^^^^^^

.. function:: magpie.coords.polar2cart(r, p[, center=[0., 0.]])

    Return cartesian coordinates for a given set of polar coordinates.

    :Parameters:
      r : array
          Radial coordinate.
      phi : array
          Phi coordinate.
      center : list
          Center point of polar coordinate grid.

    :Returns:
      x : array
          x coordinate.
      y : array
          y coordinate.
