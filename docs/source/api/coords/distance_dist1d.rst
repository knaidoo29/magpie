
magpie.coords.dist1d
^^^^^^^^^^^^^^^^^^^^

.. function:: magpie.coords.dist1d(x1, x2)

    Returns the distance between two points in 1D.

    :Parameters:
      x1, x2 : float or array
          Two points for which the distance is required.

    :Returns:
      dist : float or array
          Distance between x1 and x2.
